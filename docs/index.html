we're g-*hic*-gonna build an irc bot, morty

Features

  1. If you send it a message that starts with 
    !jsrnbot say
    it'll read it out in my living room through a raspberry pi with a scottish accent.
  2. If you say 
    !jsrnbot help
    it'll tell you to visit this page.
  3. If it detects a youtube link getting pasted with no context, it'll fetch the title and channel name.
